Sitio web creado para la fundación **Patitas Felices**, enfocado en promover la adopción de mascotas y facilitar la comunicación con la organización. El proyecto destaca por su excelente **estructura semántica en HTML** y su diseño visual atractivo, desarrollado puramente con HTML y CSS, sin el uso de JavaScript.

🚀 **Demo en vivo:** [Patitas Felices en Vercel](https://patitas-felices-blond.vercel.app)

## ✨ Características Principales

- 🏠 **Página Principal (`index.html`):** Presentación de la fundación, su misión y visión.
- 🐶 **Galería de Adopción (`catalogo-animales.html`):** Un catálogo visual estructurado con las mascotas que buscan un hogar.
- 🐾 **Perfiles Detallados (`perfil-ejemplo-pipo.html`):** Vistas individuales para conocer la historia y características de animales específicos, como Pipo.
- ✉️ **Contacto (`formulario-contacto.html`):** Formulario semántico para consultas y solicitudes de información.

## 🛠️ Tecnologías

- **HTML5 (75%):** Estructura de la información de manera accesible y semántica.
- **CSS3 (25%):** Estilos, colores y disposición (basada en el diseño de wireframes) para una interfaz limpia.
